Outlook Recurring Email Scheduler

A Python script that automatically sends recurring emails through Outlook at scheduled times. Uses pywin32 and schedule for automation.

Instructions for Full Automation
  1. Open Task Scheduler
  2. Create Basic Task
  3. Trigger: At log on or Daily
  4. Action: Start a Program
  5. Program: python.exe
  6. Arguments: C:\path\to\outlook_scheduler.py
  7. Start in: C:\path\to\
Scheduling Options & Adjustments

You can adjust when the script sends emails and to whom:

  • Daily at 9:00 AM: schedule.every().day.at("09:00").do(send_outlook_email)
  • Weekly (Monday at 10:30): schedule.every().monday.at("10:30").do(send_outlook_email)
  • Every 2 hours: schedule.every(2).hours.do(send_outlook_email)
  • Multiple recipients: mail.To = "a@example.com; b@example.com"
