Outlook Recurring Email Scheduler

A Python script that automatically sends recurring emails through Outlook at scheduled times. It uses pywin32 and schedule to automate routine outreach while keeping timing flexible.

Instructions for full automation
  1. Open Task Scheduler.
  2. Create a basic task.
  3. Choose a trigger such as At log on or Daily.
  4. Select Start a program as the action.
  5. Set the program to python.exe.
  6. Provide C:\path\to\outlook_scheduler.py as the argument.
  7. Set C:\path\to\ as the start-in directory.
Scheduling options and adjustments

You can adjust when the script sends emails and who receives them:

  • Daily at 9:00 AM: schedule.every().day.at("09:00").do(send_outlook_email)
  • Weekly: schedule.every().monday.at("10:30").do(send_outlook_email)
  • Every 2 hours: schedule.every(2).hours.do(send_outlook_email)
  • Multiple recipients: mail.To = "a@example.com; b@example.com"
