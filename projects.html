Outlook Recurring Email Scheduler

A Python script that automatically sends recurring emails through Outlook at scheduled times. Uses pywin32 and schedule for automation.

Instructions for Full Automation

To make this fully automated using Windows Task Scheduler: 1. Open Task Scheduler 2. Create Basic Task 3. Trigger: At log on or Daily 4. Action: Start a Program 5. Program: python.exe 6. Arguments: "C:\path\to\outlook_scheduler.py" 7. Start in: C:\path\to\
